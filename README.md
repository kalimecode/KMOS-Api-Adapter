# KMOS-Api-Adapter
Универсальный программный шлюз, предназначенный для интеграции разнородных промышленных устройств (датчики температуры, блоки газоанализа БГЕ, видеокамеры) в единый стандартизированный HTTP/JSON-интерфейс.
(Необходимо подключение блока ЭССА!)

=====================================
= Быстрый старт
=====================================
Доступ к веб-дашборду
Откройте в браузере:
http://<IP-адрес-сервера>:81/

## Получить все данные одним запросом
curl -s http://localhost:81/api/getAll | jq .
# Температура с датчиков

curl -s http://localhost:81/api/getSensors | jq .

# Данные блока БГЕ
curl -s http://localhost:81/api/getBGE | jq .

# Информация о камере
curl -s http://localhost:81/api/camera | jq .

# Статус системы
curl -s http://localhost:81/api/status | jq .

# Скачать изображение с камеры
curl -O http://localhost:81/camera/lastest.png
